## 已知边界

- 该项目不是医疗器械，不能用 PIR/称重状态机替代服药确认、医嘱、护理、诊断或紧急处置；
- 固件 AP/STA 和 REST 都使用无认证、无 TLS 的 HTTP，且 CORS 开放；仅限隔离可信局域网；
- App 的一次状态 HTTP 响应不证明设备在线、传感器准确、提醒送达或药物已经被服用；
- `SmartPillbox` 是源代码里的公开开放式教学 AP 名称，不是安全网络凭据；
- OLED 实现文件存在，但当前 `main.cpp` 没有调用 `displayBegin()` 或 `displayTick()`；
- 状态机阈值是 HX711 原始计数参数，未有当前实物标定、误报率、漏报率、剂量准确度、功耗、网络可靠性或长期稳定性证据；
- 不得在完成日期化、精确 commit 绑定的真机复测前使用“系统在线”“硬件已验证”“稳定运行”“提醒可靠”“服药已确认”“医疗可用”或“生产就绪”等表述。
